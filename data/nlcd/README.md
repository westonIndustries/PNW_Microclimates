# NLCD Imperviousness

Place the NLCD impervious surface GeoTIFF here.

**Expected file**: `nlcd_impervious_2021.tif`  
**Config constant**: `NLCD_IMPERVIOUS_RASTER`  
**Source**: USGS Multi-Resolution Land Characteristics Consortium (https://www.mrlc.gov)  
**Resolution**: 30 m  
**Values**: 0–100 (percent impervious); sentinel values 127 and 255 are treated as nodata
