# MesoWest Wind Observations

Place per-station wind observation CSV files here, one file per station.

**Expected files**: `<STATION_ID>.csv` (e.g., `KPDX.csv`, `KEUG.csv`)  
**Config constant**: `MESOWEST_WIND_DIR`  
**Source**: MesoWest / Synoptic Data (https://synopticdata.com)  
**Required columns**: `date_time`, `wind_speed_set_1` (m/s)  
**Coverage**: Annual observations used to compute mean and 90th-percentile wind speed per station
